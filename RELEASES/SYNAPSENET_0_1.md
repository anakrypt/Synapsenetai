# SYNAPSENET 0.1

## Release
- Version: `0.1.0-beta`
- Project: SynapseNet / KeplerSynapseNet
- Repository root: `Synapsenet-main`

## Summary
SynapseNet 0.1 provides a local-first decentralized intelligence node with:
- PoE-based knowledge contribution and validation flows
- NGT wallet/ledger/transfer foundations
- Node-attached agent runtime (NAAN)
- TUI-based operations and diagnostics
- Optional web context injection (clearnet/onion) with Tor-first routing controls

## Included Areas
- Core consensus, ledger, wallet, transfer
- Agent runtime/scheduler/coordination/score/storage pipeline
- Tor bridge/provider/process-guard/route-policy components
- Privacy modules (onion service, SOCKS5, dandelion, traffic controls)
- Model loader/inference/marketplace foundations
- Web connector stack (search, extraction, audit, RPC)
- Cross-platform Tor validation and automation runbooks/scripts

## Notes
- This is a beta release snapshot and may change quickly.
- Operator should use runbooks under `KeplerSynapseNet/docs/` for Tor and validation workflows.
- Architecture/specification source of truth remains in `interfaces txt/`.
